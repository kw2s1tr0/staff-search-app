# プロジェクト概要

このリポジトリでは、Laravel の学習・練習を目的として社員検索サイトを開発しています。

## 技術構成

- Laravel
- Blade
- htmx
- Tailwind CSS

## ドメイン前提

- 社員データは、本来は外部DBに存在する想定です。
- この練習アプリでは便宜上マイグレーションやSeederで社員テーブルを用意していますが、アプリの主目的は社員データの検索・参照です。
- 社員データの作成、更新、削除は主目的ではありません。
- 社員APIを作る場合は、原則として一覧・検索用の `index` と詳細表示用の `show` を中心に設計します。
